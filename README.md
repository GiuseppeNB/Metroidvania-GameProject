# Como jogar o jogo
O jogo deve ser executado pelo VSCode e a extensão Live Server tem que baixar pois o jogo possui mais de um arquivo e, por conta disso, não pode ser aberto somente com um duplo-clique no arquivo HTML pois devido a segurança do navegador, o CORS fará com que ele não possa rodar.
O jogo consiste em o usuário tentar achar a saída da fábrica para escapar de lá sem que seja atingido por um robô inimigo. Durante o percurso, o usuário pode pegar recuperadores de vida e, para chegar ao final do jogo, o usuário deve lutar com o robô "Boss", onde vai desbloquear o pulo-duplo para que consiga chegar até a saída. Para movimentar o personagem deve-se utilizar as setas direita e esquerda, para pular deve-se pressionar a tecla "x" e para atacar deve-se pressionar a tecla "z".
